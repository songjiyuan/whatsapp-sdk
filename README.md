# whatsapp-sdk-nodejs

纯协议版sdk，提供socket接口，可实现单机高并发高负载。

功能方面，已经完成注册登陆、上传通讯录、拉群加群、发消息给联系人、发消息到群等基本功能。

whatsapp版本：2.21.5.18 安卓版 （2021 年 3 月 19 日更新）

可根据此sdk实现第三方whatsapp客户端（使用手机验证码登陆，非web版本）。
